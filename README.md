# 📡 محوّل شفرة مورس - Ahmed El-aref

تطبيق أندرويد احترافي لتحويل النصوص إلى شفرة مورس والعكس، مع دعم كامل للغة العربية والإنجليزية والرموز.

---

## 🚀 خطوات فتح المشروع في Android Studio

1. افتح **Android Studio** (Arctic Fox أو أحدث)
2. اختر **Open** → ثم حدد مجلد `MorseApp`
3. انتظر حتى يكتمل تحميل Gradle
4. اضغط **Run ▶** لتشغيل التطبيق على جهاز أو محاكي

## ✨ مميزات التطبيق

### 🌍 دعم اللغات
- ✅ جميع الحروف العربية (ا ب ت ث ج ح خ د ذ ر ز س ش ص ض ط ظ ع غ ف ق ك ل م ن ه و ي)
- ✅ الحروف الخاصة: أ إ آ ة ى ء ئ ؤ
- ✅ الحروف الإنجليزية (A-Z)
- ✅ الأرقام (0-9)
- ✅ الرموز (. , ? ! : ; ' " ( ) / @ & = + - _)

### 🔐 نظام التشفير
- قاموس مورس مخصص عربي كامل (ITU)
- فصل الحروف بمسافة
- فصل الكلمات بـ /
- استبدال الحروف غير المعروفة بـ ?

### 🔊 الصوت والاهتزاز
- تشغيل صوت المورس (نقطة = قصير، شرطة = طويل)
- تحكم في السرعة (5-30 كلمة/دقيقة)
- تحكم في مستوى الصوت
- اهتزاز اختياري مع المورس

### 🎨 التصميم
- Dark Mode افتراضي
- Material Design 3
- تصميم متدرج احترافي
- أنيميشن عند التحويل

### ⚙️ وظائف إضافية
- كشف تلقائي للغة
- نسخ النتيجة للحافظة
- مشاركة مباشرة (واتساب، تيليجرام)
- يعمل بدون إنترنت 100%
